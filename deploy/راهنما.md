## 🚀 نصب سریع (۱۰ دقیقه)

### پیش‌نیازها
1. Ubuntu Server 22.04 LTS نصب شده
2. دسترسی SSH یا terminal
3. کارت وایرلس با `AP mode`

### گام ۱ — کپی فایل‌ها به سرور
```bash
# از کامپیوتر ویندوز:
scp -r deploy/ ubuntu@IP_SERVER:/tmp/wifi-media-deploy/

# یا با USB flash drive، کپی پوشه deploy به /tmp/wifi-media-deploy/
```

### گام ۲ — نصب
```bash
cd /tmp/wifi-media-deploy
chmod +x install.sh

# با تنظیمات پیش‌فرض:
sudo bash install.sh

# یا با تنظیمات سفارشی:
sudo WIFI_IFACE=wlan1 WIFI_SSID="مسافر ترین" WIFI_PASS=train2024 bash install.sh
```

### گام ۳ — کپی فایل‌های رسانه
```bash
# کپی فیلم‌ها به /mnt/media
mkdir -p /mnt/media/movies /mnt/media/series /mnt/media/music
cp /path/to/movies/* /mnt/media/movies/

# یا mount یک هارد اکسترنال:
sudo mkdir /mnt/media
sudo mount /dev/sda1 /mnt/media
# برای mount دائمی در /etc/fstab اضافه کنید
```

### گام ۴ — Reboot
```bash
sudo reboot
```

### گام ۵ — تست
```
📱 گوشی خود را به شبکه WiFi-Media وصل کنید
🌐 مرورگر را باز کنید → خودکار به صفحه اصلی redirect می‌شوید
```

## ⚙️ متغیرهای تنظیم install.sh

| متغیر | پیش‌فرض | توضیح |
|-------|---------|-------|
| `WIFI_IFACE` | `wlan0` | نام کارت وایرلس |
| `ETH_IFACE` | `eth0` | کارت اترنت |
| `WIFI_SSID` | `WiFi-Media` | نام شبکه WiFi |
| `WIFI_PASS` | `wifimedia2024` | رمز WiFi |
| `SERVER_IP` | `192.168.100.1` | IP سرور |
| `DEPLOY_DIR` | `/opt/wifi-media` | مسیر deploy |

### مثال برای قطار:
```bash
sudo WIFI_SSID="قطار رجا - کوپه ۳" \
     WIFI_PASS="raja2024" \
     SERVER_IP="10.0.0.1" \
     bash install.sh
```

### مثال برای هتل:
```bash
sudo WIFI_SSID="Hotel Entertainment" \
     WIFI_PASS="hotel1234" \
     WIFI_IFACE=wlan1 \
     bash install.sh
```

## 🔧 دستورات مفید بعد از نصب

### وضعیت سرویس‌ها
```bash
# بررسی همه سرویس‌ها
sudo systemctl status wifi-media hostapd dnsmasq nginx

# بررسی Docker
sudo docker compose -f /opt/wifi-media/docker-compose.yml ps

# لاگ‌ها
sudo journalctl -u wifi-media -f
sudo journalctl -u hostapd -f
sudo docker logs wifi-media-backend -f
```

### کاربران متصل
```bash
# لیست IP های DHCP
cat /var/lib/dnsmasq/dnsmasq.leases

# ترافیک شبکه
sudo iftop -i wlan0

# تعداد کاربر
iw dev wlan0 station dump | grep Station | wc -l
```

### اسکن مدیا
```bash
# اسکن دستی
curl http://localhost:4000/api/media/scan -X POST \
     -H "Authorization: Bearer ADMIN_TOKEN"
```

### تغییر رمز ادمین
```bash
# وارد container شوید
sudo docker exec -it wifi-media-backend sh
# در داخل container:
node -e "
const db = require('better-sqlite3')('/data/wifi-media.db');
const bcrypt = require('bcryptjs');
const hash = bcrypt.hashSync('NEW_PASSWORD', 10);
db.prepare('UPDATE users SET password=? WHERE username=?').run(hash,'admin');
console.log('✓ رمز تغییر کرد');
"
```

## 🌍 Captive Portal — چطور کار می‌کند؟

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  📱 گوشی کاربر                                          │
│      ↓ (متصل به WiFi-Media)                             │
│      ↓ HTTP request به هر سایتی                         │
│                                                         │
│  🔀 iptables PREROUTING                                 │
│      → همه HTTP (port 80) را به 192.168.100.1 برمی‌گرداند │
│                                                         │
│  🌐 nginx (روی 192.168.100.1:80)                        │
│      → /generate_204 → 302 redirect (Android)          │
│      → /hotspot-detect.html → 302 (Apple)               │
│      → بقیه → proxy به WiFi-Media App                  │
│                                                         │
│  📺 WiFi-Media App (port 8181)                         │
│      → صفحه login → Home → تماشای محتوا                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## ❓ مشکلات رایج

### `hostapd` start نمی‌شود
```bash
# بررسی لاگ
sudo systemctl status hostapd
sudo journalctl -u hostapd --no-pager | tail -20

# راه‌حل: غیرفعال کردن NetworkManager
sudo systemctl stop NetworkManager
sudo systemctl disable NetworkManager
```

### کاربران IP نمی‌گیرند
```bash
# بررسی dnsmasq
sudo systemctl status dnsmasq
# بررسی port 67
sudo ss -ulnp | grep :67
# اگر busy بود:
sudo systemctl stop systemd-networkd
```

### Captive Portal نمایش داده نمی‌شود
```bash
# بررسی iptables
sudo iptables -t nat -L PREROUTING -n -v
# اجرای مجدد
sudo bash /opt/wifi-media-iptables.sh
```

### Docker start نمی‌شود
```bash
sudo systemctl restart docker
sudo docker compose -f /opt/wifi-media/docker-compose.yml up -d
```
